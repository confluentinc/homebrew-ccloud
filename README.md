# homebrew-ccloud

A repository for Confluent Cloud related brews.

## Requirements

* [Homebrew](https://github.com/Homebrew/brew)
* Java (>= 7)

## Installation

Run the following in your command-line:

```sh
$ brew tap confluentinc/ccloud
```

Then to install `ccloud` run:

```sh
$ brew install ccloud
```
